Open Node-RED's Palette Manager:

    Click the menu button in the top-right corner of the Node-RED interface.​
    Select "Manage Palette" from the dropdown menu.​

Install the Dashboard:

    Navigate to the "Install" tab within the Palette Manager.​
    In the search bar, type @flowfuse/node-red-dashboard.​
    Locate the package named @flowfuse/node-red-dashboard and click the "Install" button next to it.​
